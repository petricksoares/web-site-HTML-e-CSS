# Sobre o Projeto
Este projeto consiste na criação de um site de portfólio pessoal, desenvolvido como atividade acadêmica da disciplina de desenvolvimento web.

O objetivo do trabalho foi aplicar na prática os conceitos de HTML semântico e posteriormente estilização com CSS, criando uma página que apresente informações profissionais de forma organizada e visualmente agradável.

O site apresenta informações básicas sobre o autor, incluindo formação acadêmica, habilidades técnicas, áreas de interesse e formas de contato
